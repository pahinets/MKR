# Event Manager System (MKR)

Система управління подіями, реалізована на Java з використанням патернів проєктування.

## Використані патерни
- **Singleton (Одинак):** Клас `EventManager`. Забезпечує існування єдиного менеджера розсилки.
- **Observer (Спостерігач):** Інтерфейс `ISubscriber`. Дозволяє модулям підписуватись на події.

## Компоненти
1. **AuthModule:** Генерує події входу та помилок.
2. **LoggerModule:** Логує всі події системи.
3. **UIModule:** Відображає повідомлення для користувача.

## Запуск
Скомпілювати та запустити з кореневої папки:
`javac MKR/*.java`
`java MKR.Main`
